Technologies and Frameworks Used in GEONEX

Convolutional Neural Networks (CNN)

Used for automatic classification of satellite images into land cover categories such as forest, river and urban areas.

Satellite Imagery

High resolution satellite images are used as input data for analysing geographical and environmental patterns.

Flask Backend

Cloud Storage

System Workflow

How GEONEX Processes Satellite Images

GEONEX follows a structured and efficient workflow to analyse satellite images using deep learning techniques, store the processed data securely and generate location based analytical reports for users.

Step 1

User Login and Data Submission

The user registers and logs into the system to gain authorised access. After authentication, the user uploads satellite images along with the corresponding location information for analysis.

  • Secure user authentication using login credentials.
  • Upload of satellite image and associated location details.
Step 2

Satellite Image Classification

The uploaded satellite image is preprocessed and passed through a convolutional neural network model, which analyses visual patterns and classifies the image into predefined land cover categories.

  • Image preprocessing such as resizing and normalisation.
  • CNN based prediction of land cover type from image features.
Step 3

Data Storage and Report Generation

After classification, the satellite image is stored in a cloud platform, while classification results and location details are saved in the database. Users can later retrieve this information and generate detailed reports.

  • Secure storage of images in the cloud with database references.
  • Location wise retrieval of data and downloadable analytical reports.
